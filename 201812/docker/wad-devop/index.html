在项目实战中应用到的开发运维

参考了庞大的k8s,2018年1月份做了个简单的devop系统，在三个项目中使用到这套东西，到目前为止运行良好。
实现了推送代码时，所部署的相应分支，会实现自动更新docker镜像。
基于 spring-cloud maven Jenkins(groovy) node python docker
另外集成了 Mojo-Weixin 和Mojo-Webqq 做运维的通知
